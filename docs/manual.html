Lua-httpd documentation

This is the documentation for the Lua-httpd package.

The networking extension implements and exports the following primitives:

bind()

Listen upon a port to accept incomming connections.

accept()

Accept a new client connection from a socket, which has previously been returned from a call to bind().

connect()

Make an outgoing TCP/IP connection to a remote host.

read()

Read data from a connected socket.

write()

Write data to a connected socket.

close()

Close an open socket.

Using the library should be simple for anybody familiar with network programming under C, and some example code is available for study.
